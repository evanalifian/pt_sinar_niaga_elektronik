# PT. Sinar Niaga Elektronik

Merupakan aplikasi berbasis ***dashboard*** untuk mengelola administrasi pada perusahaan **Sinar Niaga Elektronik**.

## Clone This Project

```bash
git clone https://github.com/evanalifian/pt_sinar_niaga_elektronik.git
git fetch origin
```

Keterangan:

- `git fetch` digunakan untuk mengambil semua ***branch*** pada remote repository.
